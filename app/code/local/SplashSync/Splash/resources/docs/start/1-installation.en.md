### Manual Installation

* Download latest stable version here : [Splash Modules](http://www.splashsync.com/en/)
* Copy all module files on your Magento app folder (app/*)
* Enable & Configure the module

### Requirements

* PHP 7.2+
* Magento 1.9.x
* An active Splash Sync User Account
